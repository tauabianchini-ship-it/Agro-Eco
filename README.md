# 🌱 Agro-Eco

Sistema desenvolvido para promover práticas agrícolas sustentáveis, auxiliando produtores rurais no monitoramento, gerenciamento e análise de informações relacionadas à produção agrícola e ao meio ambiente.

## 📋 Sobre o Projeto

O Agro-Eco tem como objetivo integrar tecnologia e sustentabilidade no setor agrícola, oferecendo ferramentas para acompanhamento de indicadores ambientais, gestão de cultivos e apoio à tomada de decisão.

## 🚀 Funcionalidades

- Cadastro e gerenciamento de propriedades rurais
- Monitoramento de culturas agrícolas
- Controle de indicadores ambientais
- Visualização de dados em dashboards
- Relatórios e análises de desempenho
- Gestão sustentável da produção

## 🛠️ Tecnologias Utilizadas

Adicione aqui as tecnologias do projeto:

### Back-end

## ⚙️ Instalação

### Pré-requisitos

- Node.js 18+
- Git
- Banco de dados configurado

### Clonando o repositório

```bash
git clone https://github.com/tauabianchini-ship-it/Agro-Eco.git
```

### Entrando na pasta do projeto

```bash
cd Agro-Eco
```

### Instalando dependências

```bash
npm install
```

### Executando o projeto

```bash
npm run dev
```

## 🔧 Configuração

Crie um arquivo `.env` na raiz do projeto:

```env
DATABASE_URL=sua_url_do_banco
PORT=3000
```
